# Superstore Data Cleaning Project

Data cleaning and analysis of Kaggle Superstore dataset using Python (pandas).

## What I did
- Cleaned the raw dataset and saved a cleaned CSV.
- Analyzed sales and profits by category and sub-category.
- Found that the Furniture category, especially Tables, was losing money.
- Made recommendations for improving profitability.

## Folder Structure
- `raw_data/` : Original dataset
- `cleaned_data/` : Cleaned CSV
- `scripts/` : Jupyter Notebook with cleaning and analysis
- `documentation/` : Report and notes

## Tools Used
- Python
- pandas
